 Vite Gourmand (ECF)

Application web PHP (PDO) + MySQL permettant de commander des menus pour des événements, 
avec gestion des rôles (admin / employé / utilisateur), suivi de commandes, annulation et gestion/modération des avis.

 Fonctionnalités:

 Côté utilisateur:
- Inscription / connexion / déconnexion
- Affichage des menus disponibles
- Consultation du détail d’un menu
- Passer une commande (date, heure, nombre de personnes, etc.)
- Suivi des commandes (page "Mes commandes")
- Annulation d’une commande avec motif
- Historique des changements de statut d’une commande
- Déposer un avis 

 Côté employé:
- Liste des commandes
- Mise à jour du statut d’une commande (suivi)
- Accès à l’historique d’une commande
- Gestion des avis (validation/refus)

 Côté admin:
- Gestion des menus (CRUD)
- Gestion des plats et association menus/plats 
- Gestion des utilisateurs/roles 
- Accès complet aux pages protégées

 Stack technique:
- PHP (PDO)
- MySQL (phpMyAdmin)
- HTML/CSS
- JavaScript 
- Serveur local : XAMPP (Apache + MySQL)

 Installation (local):

 Prérequis:
- XAMPP installé
- PHP 8.x 
- MySQL/MariaDB via phpMyAdmin
